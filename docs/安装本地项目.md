# 安装本地项目

## 通过`poetry`

该过程会将项目依赖和项目本身一同安装到当前环境中，安装的项目可以动态编辑

```shell
poetry install
```

## 通过`pip`

该过程按照的项目同样可以动态编辑

```shell
pip install --editable .
```

## 通过`pip`（生产环境）

1. 构建软件

    构建产出位于`dist`文件夹中

    ```shell
    poetry build
    ```

2. 安装软件包

    文件名根据实际情况会有所变动

    ```shell
    cd dist
    pip install openvault-0.1.0.tar.gz
    ```

## 卸载软件包

可以通过`pip`卸载

```shell
pip uninstall openvault
```
